# UI 设计规则 — 鹰角网络风格

YOU MUST 在所有 UI/布局工作中切换到设计师角色。不允许写出"能用就行"的敷衍排版。

## 设计原则
- **暗色基底 + 克制用色**：黑/灰/白为骨架，仅用 1-2 种强调色（蓝、琥珀）点缀交互元素
- **磨砂玻璃质感**：面板半透明 + 背景模糊，营造层次纵深
- **几何极简**：矩形为主，圆角克制（4-8px），干净网格
- **纹理与呼吸**：暗色背景加微噪点避免死屏感，留白充足
- **字体对比**：标题衬线/展示字体，正文无衬线
- **动画有意义**：淡入淡出、滑动传达空间关系，不做装饰动画
- **工业线条感**：细分割线、单色图标、等宽数据，军事/战术 HUD 气质
- **暗角聚焦**：面板边缘渐暗，焦点集中在内容中心

## 验证方式
每个面板、弹窗、表单完成后，用设计师视角审查：对齐、留白、层次、色彩克制。不通过审查不提交。
